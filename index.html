Politique de confidentialité — CardScan Pro

Dernière mise à jour : mars 2026

1. Collecte de données

CardScan Pro ne collecte aucune donnée personnelle. Toutes les données (collection de cartes, decks, préférences) sont stockées uniquement sur votre appareil et ne sont jamais transmises à nos serveurs.

2. APIs tierces

L'application effectue des requêtes vers des APIs tierces pour récupérer les informations sur les cartes et les prix :

  • Scryfall API (Magic: The Gathering)
  • Pokemon TCG API (Pokémon)
  • OPTCG API (One Piece Card Game)
  • JustTCG API (prix Cardmarket)

Ces requêtes ne contiennent aucune information personnelle. Aucun compte utilisateur n'est requis.

3. Permissions

  • Caméra : uniquement pour scanner vos cartes
  • Stockage : pour sauvegarder votre collection localement

4. Partage de données

Nous ne vendons, ne louons et ne partageons aucune donnée avec des tiers.

5. Sécurité

Toutes les communications avec les APIs tierces sont chiffrées via HTTPS.

6. Enfants

Cette application n'est pas destinée aux enfants de moins de 13 ans.

7. Contact

Pour toute question : mathis.ce@hotmail.fr

8. Modifications

Cette politique peut être mise à jour. La date de mise à jour est indiquée en haut de ce document.
